Marcação de texto

Podemos criar também um texto marcado usando a tag MARK

Logo abaixo podemos visualizar uma repetição de marcação com a mesma tag

Texto grande e pequeno

Estamos criando um texto pequeno

Para criarmos um texto grande, podemos utilizar a tag <big>, porém ela já está obsoleto pela W3C e o ideal para alterar essa forma é utilizando o CSS

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois, utilizando a tag <ins>(semântico)

Existe também o sublinhado com a tag <u>, porém é um termo não-semântico

Texto sobrescrito

Para inserir coisas do tipo x20+3, utilize a tag <sup>

Texto subscrito

Para inserir coisas do tipo H20, utilize a tag <sub>
